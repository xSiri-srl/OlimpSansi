# TisOlimpSansi11
##
 Iniciar FrontEnd

1. Redirigirse a la carpeta TisOlimSansi_FrontEND
```sh
cd TisOlimpSansi_FrontEnd
```

2. Descargar dependencias
```sh
npm install
```

3. Hacer correr el servidor FrontEnd
```sh
npm run dev
```
##
